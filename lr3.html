1. Ключевой прототип дипломного проекта

Ключевым прототипом дипломного проекта явлется схема расположения ТД в БЛВС и модуль безопасности для данной сети.

Данный прототип является:

  • По назначению: вертикальный

  • По времени жизни: эволюционный

  • По способу реализации: электронный/бумажный



    • 2. Прикладная задача и основные требования к личной SPA

    При выполнении данного проекта было принято решение о конкретизации задач и сроков исполнения работ при помощи календаря, написанного на фреймворке Angular JS. Важно отметить, что существует библиотека "angular-calendar", функционал которой необходимо переработать под себя и встроить в созданные ранее Github Page's. Критерием выполнения личной SPA можно считать Task-calendar проекта по созданию справочной системы, представляющей существующую инфраструктуру БЛВС в зданиях Университета.



    3. Пользовательская история

    Как пользователь справочной системы я хочу получить графическое отображение карты расположения точек доступа с целью большего понимания и полной наглядности.



    4. Классификация планируемого прототипа выбранной для коллективной разработки задачи:

  • По назначению: вертикальный

  • По времени жизни: одноразовый

  • По способу реализации: электронный



    • 5. Прикладная задача для реализации в модели MVC, фреймворк и проект, в котором она будет решаться

  • Задача: создание справочной системы для получения информации о точках доступа в существующей инфраструкуре сети

  • Используемый фреймворк: Django - свободный фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC.

  • Проект: задача для лабораторных 5 - 7
